Design Toolkit Pattern Portfolio Introduction

Introduction

The pattern portfolio is a collection of reusable elements and componenets that can be used to create page layouts.

There are code snippets and Markdown examples that can be used for reference or pasted in to the CMS. There are also several helper classes and tools that can be used to acheive complex layouts.

Additionally, there is documentation about code guidelines and scaffolding to help with developer onboarding

Web Typography Styles for basic text elements and components
Buttons All the flavors
Forms Input types and best practices
Images Sizes, placement, and use cases
Modules Components large and small
Developers In-depth docs for onboarding and maintaining the design system
